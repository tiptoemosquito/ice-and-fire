
  ICE and Fire 
================
   A GoT CLI
----------------

**Installation**
----------------
 
 run ruby bin/got

**Description**
---------------

This CLI provides a list of characters from the extremely popular books and HBO show, _Game of Thrones_, per the An Api of Ice and Fire website.
